Troubleshooting tips

This website is in its early days. Some tables might not parse correctly, so try doing the following to fix up the table formatting:

  • Check whether each row is in the format label value value value, and that the label only occupies one line. Some labels might be copied and pasted across multiple lines and right now the program isn't built to handle this.

  • Remove headers. I am working on additional functionality to handle the headers, but right now the entire row will copy and paste as the first label in a row.

  • Provide feedback for future updates here.
